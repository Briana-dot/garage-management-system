# Garage Management System

A web-based Garage Management System developed using PHP, MySQL, CSS, and JavaScript.

## Features

* Vehicle Registration
* Customer Management
* Service Booking
* Payment Tracking
* Dashboard Analytics
* Parking Management
* Staff Management

## Technologies

* PHP
* MySQL
* CSS
* JavaScript

🚧 Under Development

This project is currently being developed and new modules are being added regularly.
